Это база данных для работы с закупками. 
Первая таблица показывает, какие заявки на закупку других структурных подразделений есть в программе развития на год. 
Вторая таблица - программа развития. 
Третья - заявки структурных подразделений.
Четвертая - структурные подразделения. 
Пользователь может редактировать со 2 по 4 таблицу включительно. Первая таблица является выборкой. 
Найти необходимое структурное подразделение можно через QComboBox "Наименование структурных подразделений", где перечислены все
подразделения в базе данных, с помощью ввода данных руководителя в соотвествующем поле, ввода номера кабинета.
Можно сократить количество подразделений в QComboBox "Наименование структурных подразделений" выбрав тип структурного подразделения.
Данные хранятся в базе данных, построенных с помощью SQLite3.
Пользовательское окно было создано с помощью PyQt5.
В коде есть документация к функциям.
